Setup & Uses

Ferramentas, extensões e configurações que fazem parte da minha rotina de desenvolvimento. Não é patrocínio — é o que realmente uso.

Editor & Terminal

  • VS Code Editor principal

    Tema escuro, fonte JetBrains Mono. Limpo, rápido e extensível para qualquer stack.

  • Windows Terminal Terminal

    Com PowerShell configurado. Uso para rodar scripts Python, comandos Git e npm.

  • Git Bash Git

    Para comandos Git do dia a dia quando estou fora do VS Code integrado.

Linguagens & Stacks

  • Python Principal

    Scripts, automações e primeiras APIs. É onde concentro a maior parte dos estudos hoje.

  • Node.js Aprendendo

    Explorando o ecossistema JS no back end. Express para APIs simples e roteamento.

  • MySQL / PostgreSQL Banco de dados

    SQL relacional para a maioria dos projetos. DBeaver como cliente visual.

Ferramentas

  • Postman APIs

    Para testar e documentar endpoints. Uso bastante durante o desenvolvimento de qualquer API.

  • GitHub Versionamento

    Controle de versão e repositório público. Issues e README como hábito desde o início.

  • Notion Organização

    Roadmap pessoal de estudos, notas de conceitos e planejamento de projetos.

  • Docker Estudando

    Ainda no início. Já consigo subir containers simples, mas preciso de mais prática.
